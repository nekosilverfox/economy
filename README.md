# 股票交易



## 头寸

“**头寸**”是金融和投资领域的一个基本术语，用来表示你在某个金融产品（如股票、期权、外汇、期货等）上的**持仓情况**。它是英文单词 **position** 的中文翻译。

---

**📌 分类举例**

| 类型            | 名称           | 举例              | 意思                     |
| --------------- | -------------- | ----------------- | ------------------------ |
| **多头头寸**    | Long Position  | 持有100股AAPL股票 | 看涨，预期价格上涨能赚钱 |
| **空头头寸**    | Short Position | 卖空1张TSLA期权   | 看跌，预期价格下跌能赚钱 |
| **平仓/无头寸** | Flat / Neutral | 已卖出所有标的    | 没有在该资产上押注涨跌   |



## SPX Delta

在证券账户中，**SPX Delta** 指的是投资组合或单个头寸对 **标普500指数（S&P 500，简称 SPX）价格变动的敏感度**，它常用于衡量股票或期权头寸的市场风险敞口。

---

**📌 什么是 Delta？**

在期权和衍生品交易中，**Delta 是一个对冲指标**，表示标的资产价格变动对期权价格影响的大小。它通常用 0 到 1 之间的数字（或 -1 到 1 之间）表示：

- **Delta = 0.5** 意味着标的资产上涨 1 点，期权价格将上涨约 0.5 点。
- **Delta = -0.3** 表示价格上涨时期权价格反而下降。

------

 🧮 SPX Delta 在账户中的含义

如果你的证券账户显示一个 **SPX Delta 值**，那代表你整个账户（或某个仓位）相对于 S&P 500 的暴露程度：

- **正值**：账户总体受益于 S&P 500 的上涨。
- **负值**：账户总体在 S&P 500 下跌时受益，可能有空头头寸或避险头寸。
- **接近 0**：账户基本与 S&P 500 的波动无关，即“市场中性”状态。

举例：

- SPX Delta = **+100**：S&P 500 每==涨== 1 点，账户理论上盈利约 $100
- SPX Delta = **-50**：S&P 500 每==跌== 1 点，账户理论上盈利约 $50

------

📊 实际用途

1. **风险管理**：判断你是否过度暴露于市场系统性风险。
2. **对冲调整**：可以通过指数期货、ETF 或期权调整你的 SPX Delta，使投资组合更稳健。
3. **市场预测与交易策略执行**：偏多或偏空时，调整 Delta 方向和幅度以契合预期。
